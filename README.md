# hello-react-rails

## Project Description

This project is connecting React on the front-end to Ruby-on-Rails on the backend. The app displays random messages contained in the database table.


## Built With

- Ruby On Rails
- React-Rails gem
- jsbundling-rails gem
- Webpack
- Redux Toolkit
- Rubocop
- Eslint
- Git


## Setup
Clone the repo and run bundle install in the command line to install the dependencies

```bash
$ git clone https://github.com/myelin0/hello-rails-react
$ cd hello-rails-react

## Install gems with:

```bash
$ bundle install
```

## Install style linters with
```bash
$ npm install
$ yarn install
```

## Run the application
``` 
$ ./bin/dev
```

### Visit the homepage of application
```
$ http://localhost:3000/
```
### Check linter errors
```
$ rubocop .
```
